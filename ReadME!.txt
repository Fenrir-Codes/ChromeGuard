ENG:


You can install the extension like this:

open the Chrome browser

type in the search bar at the top: chrome://extensions and press enter

At the top right there is a slider called Developer Mode, turn it on

Then click on the Load unpacked items button at the top left

select the ChromeGuard folder and click OK

Then the setup screen should appear where you can enter your password

After you have entered your password, the browser will close and can only be unlocked with a password!

Uninstall

In the same way, on the extensions page, click on the Remove extension button
